CREATIVE FOUNDATIONS 8th & 9th June 2026

Design Workshop : Layouts & Design Tools

Learn the fundamentals of modern poster design, including typography, color palettes, layout composition, and visual hierarchy. This hands-on workshop is perfect for beginners looking to create stunning posters, event banners, presentation decks, and social media creatives using Canva.

Workshop Curriculum

Understand layout grids, alignment rules, typography selection, visual hierarchies, and generating professional color schemes matching digital standards.

Introduction to Canva: vector shaping, frames, auto-layout, component states, libraries, and establishing a consistent design system.

Design professional posters from scratch, create visually engaging layouts, configure creative design elements, and produce high-quality, share-ready graphics using Canva.

MATHEMATICAL COMPUTATION 12th & 13th June 2026

MATLAB Workshop: Numeric Scripting & Simulators

Learn MATLAB scripting, data structures, and mathematical modeling methodologies. Specifically tailored for engineering problem solving, process optimization, and custom plotting applications under the guidance of Dr. Sangram Roy.

Workshop Curriculum

Workspace orientation, matrix arrays creation, linear algebra operations, indexing structures, scripting file structure, and debugging workflows.

Generating complex plots, 3D surface charts, contours, importing external experimental datasets, curve fitting tools, and custom visual formatting.

Solving Ordinary Differential Equations using solvers (ode45), setting up continuous loop dynamics, and an overview of Simulink diagrams.

FINITE ELEMENT ANALYSIS 17th February 2025

ANSYS Workshop: Structural FEA & Fluent CFD

Gain hands-on experience with ANSYS by learning CAD import, geometry preparation, mesh generation, boundary condition configuration, fluid and structural simulations, thermal analysis, and result visualization in a modern CAD environment.

Workshop Curriculum

Using SpaceClaim to edit CAD geometries, setup physical bounds, apply mechanical mesh sizes, grid refinement parameters, and check elements' qualities.

Applying support constraints, loading pressures, analyzing deformation fields, calculating equivalent Von-Mises stresses, and checking factors of safety.

Setting boundary velocity conditions, setting fluid properties, configuring K-epsilon turbulence solvers, inspecting velocity vector profiles and streamline trajectories.
